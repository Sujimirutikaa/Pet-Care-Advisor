# Pet Care Advisor - Knowledge-Based AI Agent

A Flask-based web application that uses knowledge-based agent principles to provide pet health assessments and care recommendations.

## Features

- **Knowledge-Based Reasoning**: Uses structured veterinary knowledge and rule-based inference
- **Species-Specific Assessment**: Tailored recommendations for dogs, cats, birds, and other pets
- **Emergency Detection**: Identifies critical symptoms requiring immediate veterinary attention
- **Confidence Scoring**: Provides confidence levels for diagnostic assessments
- **Treatment Recommendations**: Suggests appropriate care based on diagnosed conditions
- **Pet-Specific Considerations**: Accounts for age, breed, and medical history
